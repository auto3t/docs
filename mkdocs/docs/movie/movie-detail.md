# Movie Detail Page

Each movie has a dedicated detail page. You can reach the detail page by clicking on the movie in the list view.

There you can see additional metadata of the movie. 

Notably, if you have configured the Target bitrate on the settings page, you can see the expected file size **Auto3T** is going to look for based on the runtime of the given movie.

### Movie Status Edit
There you can manually edit the status of the movie. Usually that should not be needed, as this should be handled automatically through the pipeline steps.

## Movie Details

Click on "show details" to expand additional config details for this movie.

- Active/inactive toggle
- Overwrite target bitrate for this movie
- Add a keyword

## Release Windows
If available you can find available release windows for that movie here with the given dates and medium. That is used to determine at what point a movie will be switched to upcoming and searching.

## Cast and Crew
**Auto3T** imports main actors and crew members. Click on the Person, to navigate to the person detail page.


## Part of Collection
If the movie is part of a collection, aka boxset, you can find the collection detail and link here.

## File Metadata
If the movie has been archived, additional file and stream metadata will show here. That is as provided by the mediaserver.

The play button opens your mediaserver at the given movie.

## Manual Search for Movie
You can manually search for the movie here.
